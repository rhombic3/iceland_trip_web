# Iceland Trip Web

这是一个用于冰岛自驾旅行的静态行程网页，基于 GitHub Pages 部署。页面整合了每日行程、景点介绍、路线地图、实时天气、高地道路开放状态、费用与旅行提示，方便在电脑和手机端快速查看。

## 主要功能

- 每日行程展示：包含时间安排、景点、住宿与交通信息
- 地图路线：支持查看每天的路线与景点位置
- 景点信息：包含简介、图片、门票/费用、注意事项和地图链接
- 实时天气：自动读取天气数据，展示每日景点天气、温度、降雨和风速
- 高地路况：自动更新关键高地路线开放状态，例如 Landmannalaugar、Mælifell、Þórsmörk、Kerlingarfjöll 和 Þakgil
- 今日模式：根据日期自动跳转到对应行程
- 移动端适配：支持手机端浏览和快捷跳转
